# Font locali

I font usati dal sito sono serviti localmente da `/public/fonts` per evitare richieste automatiche a Google Fonts prima del consenso dell'utente.

- Fraunces: font open source distribuito tramite Google Fonts.
- Manrope: font open source distribuito tramite Google Fonts.

Se vengono aggiunti nuovi pesi, subset o famiglie, aggiornare anche `src/styles/global.css`.
